# Supply Chain Optimization using Big Data Analytics

## Objective
Predict shipment delays and identify key factors affecting delivery performance.

## Dataset
E-commerce Shipping Dataset (10,999 records)

## Approach
- Data preprocessing
- Exploratory data analysis
- KPI analysis
- Random Forest classification
- Feature importance analysis

## Results
- Accuracy: 66.52%
- Major delay factors:
  - Product weight
  - Product cost
